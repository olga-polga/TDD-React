## Synopsis

Minimal React js app using Mocha + chai + enzyme for testing

## Prerequisites:

install `node` and `npm` locally

## Installation
```
npm install babel-preset-react -D
```

## Run locally (http://localhost:8080/)
```
npm run dev
```

## Test
```
npm test
```

## Build + Deploy to PCF  (edit manifest.yml as needed)
```
npm run deploy
```
